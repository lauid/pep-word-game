# PEP 英语单词连连看 · 实施方案

> 目标人群：小学 3~5 年级学生 / 家长 / 英语老师
> 核心玩法：英文 ↔ 中文连连看（带 ≤2 拐角路径判定 + 英文发音）
> 技术栈：纯 HTML + CSS + JavaScript，零依赖、零联网、双击即玩

---

## 一、当前已实现

| 模块 | 说明 |
|------|------|
| 词库 `words.js` | 人教版 PEP 三起版，3~5 年级上下册共 6 册、36 单元、约 460 词，按「年级→册→单元」结构化 |
| 设置面板 | 选年级 / 选上册下册 / 多选单元 / 选难度（6/10/14对/全部） |
| 连连看核心 | 经典路径算法：两格之间是否存在「不超过 2 个拐角」的空路，存在才允许消除 |
| 发音 | 浏览器原生 `SpeechSynthesis`（en-US），点英文格或消除时朗读，可一键开关 |
| 辅助 | 提示（高亮一组可消除词）、洗牌（无可消时自动洗牌）、计时、计分 |
| 结算 | 通关弹窗：用时 / 得分 / 提示次数 / 洗牌次数，可再来一局或返回菜单 |

文件清单（同一文件夹，建议整体拷贝）：
- `index.html` —— 游戏本体（已内联样式与逻辑，引用同目录 `words.js`）
- `words.js`   —— 单词数据，改词 / 加词只动这里
- `PLAN.md`    —— 本方案

## 二、目录与运行

```
pep-word-game/
├─ index.html      # 双击用浏览器打开即可，无需服务器
├─ words.js        # 词库（可在任意文本编辑器里增删改）
└─ PLAN.md         # 实施方案
```

运行方式（任选其一）：
1. **最简单**：直接双击 `index.html`（Chrome / Edge / Safari 均可）。
2. 局域网共享：在文件夹内执行 `python3 -m http.server 8080`，同网设备访问 `http://本机IP:8080`。
3. 课堂投屏：浏览器全屏（F11）+ 触屏/鼠标均可点击。

> 发音依赖浏览器内置语音引擎；若某设备无英文语音包，游戏仍可正常玩，只是不朗读。

## 三、连连看算法要点（核心难点）

棋盘外围留 1 圈空格（边框），用于让路径「绕到外面」。
判定函数 `findPath(grid, a, b)` 用 BFS，状态为 `(行, 列, 方向, 拐角数)`：
- 从 a 向 4 个方向出发，第一线段拐角数 = 0；
- 直行不增加拐角，转向 +1；
- 拐角数 > 2 剪枝；
- 到达 b 即返回路径（用于画连线动画），否则返回 `null`。
已用 Node 单元测试验证：直线、绕行、被完全封死三种情形均正确。

## 四、其他好玩的单词游戏（可依次扩展）

同一份 `words.js` 词库可复用于以下所有玩法，建议按学生兴趣轮换：

| 玩法 | 形式 | 训练点 | 实现难度 |
|------|------|--------|----------|
| 🎧 听音选词 | 播放英文发音，从 4 张图/词中选对的 | 听力辨词 | 低（复用发音） |
| ✍️ 拼写闯关 | 给中文 + 首字母，补全拼写 | 拼写记忆 | 低 |
| 🃏 闪卡记忆 | 卡片翻面 英→中，支持「标错词本」 | 识记 | 低 |
| 🔤 单词接龙 | 上一词尾字母 = 下一词首字母 | 词汇激活 | 中 |
| 🖼️ 看图识词 | 图（emoji/图片）选对应英文 | 图文对应 | 中（需配图） |
| ⏱️ 限时挑战 | 60 秒内尽可能多配对，排行榜 | 反应速度 | 中 |
| 🗺️ 闯关模式 | 单元=关卡，三星评级 + 错词回顾 | 成就感 | 中高 |
| 👥 双人对战 | 同屏分边抢答，计分对战 | 竞争趣味 | 高 |

推荐路线图：
1. 先上线「连连看」（已完成）。
2. 加「闪卡 + 错词本」沉淀复习数据（localStorage 存错词）。
3. 加「听音选词」「拼写闯关」利用已有发音与词库。
4. 最后做「闯关 + 排行 + 双人对战」提升粘性。

## 五、后续可优化点

- **错词本**：把连续答错/提示的词存进 `localStorage`，下次优先复习。
- **2024 新版词库**：可另建 `words_2024.js` 并在设置面板加「教材版本」切换。
- **配图**：动物/食物等用 emoji 作图标，降低低龄学生阅读压力。
- **语音评测**：接入 Web Speech API 的 `SpeechRecognition` 做「跟读打分」。
- **教师后台**：导入自定义词表（Excel/CSV → JSON），适配校内补充词汇。
- **移动端适配**：当前用固定格宽；可改为按屏宽自适应 `var(--cell)`。

## 六、如何加单词（示例）

编辑 `words.js`，在对应单元 `words` 数组追加 `["english","中文"]` 即可，例如给三年级上册 Unit 1 加 `pen`：

```js
{ unit: "Unit 1 Hello!", words: [
  ["ruler","尺子"], ["pen","钢笔"],  // ← 加在这里
  ...
]}
```

游戏会在下次「开始游戏」时自动纳入，无需改其他代码。
